Overview
This dataset contains sensor-based data for pipeline fault detection, used to train and evaluate machine learning models based on a Swarm AI classification approach.

Files Included
sanjeevdata
Contains the main dataset with sensor readings representing pipeline operating conditions (normal and faulty states).
data.m
MATLAB script used for loading and preprocessing the dataset, including data preparation for model training and testing.

Data Characteristics
Sensor-derived features from pipeline systems
Includes labeled instances for fault and non-fault conditions
Suitable for supervised learning and classification tasks
